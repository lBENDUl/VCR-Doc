# Herramientas para File Inclusion

Herramientas automatizadas para detectar y explotar vulnerabilidades de tipo **Local File Inclusion (LFI)** y **Remote File Inclusion (RFI)**.

---

## Herramientas disponibles

| Herramienta | Repositorio |
|---|---|
| LFISuite | [github.com/D35m0nd142/LFISuite](https://github.com/D35m0nd142/LFISuite) |
| LFiFreak | [github.com/OsandaMalith/LFiFreak](https://github.com/OsandaMalith/LFiFreak) |
| liffy | [github.com/mzfr/liffy](https://github.com/mzfr/liffy) |

> ⚠️ Estas herramientas están escritas en Python 2 y pueden estar desactualizadas. En la mayoría de los casos es más fiable y didáctico analizar la vulnerabilidad manualmente. Consulta la guía de [LFI](../03-explotacion-web/local-file-inclusion-lfi.md) para el proceso manual.
